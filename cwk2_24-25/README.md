```ghc cwk2_24-25.hs```
2. Execute te generated file:
```./cwk2_24-25```

## Load File into GHCi (Interactive Mode):
1. Open the terminal and start GHCi:
    ```ghci```
2. Load the file into GHCi:
    ```:l cwk2_24-25.hs```
3. Run the `main` function or any other defined function:
    ```ghci>main```
    ```ghci>tribonacci 7```
    ```ghci>lazyCatererSeq 6```
    ```ghci>pretty myHorse```
